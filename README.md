# Curs4 Frontend

React приложение для работы с бэкендом Curs4.

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Приложение будет доступно по адресу http://localhost:3000

## Структура

- `/src/components/Auth` - компоненты авторизации и регистрации
- `/src/components/Dashboard` - панели для разных ролей (клиент, админ, водитель)
- `/src/services/api.js` - API сервис для работы с бэкендом
- `/src/context/AuthContext.js` - контекст аутентификации

## Роли

- **CLIENT** - клиент может управлять своими декларациями, платежами и профилем
- **ADMIN** - администратор может управлять всеми пользователями, декларациями и платежами
- **DRIVER** - водитель (базовая панель)
